Ayuda

Para crear un nuevo gráfico, sigue estos pasos:

  1. Haz clic en "Nuevo Gráfico" en el menú lateral
  2. Selecciona el tipo de gráfico que deseas crear
  3. Elige el modelo de datos y los campos para los ejes X e Y
  4. Configura opciones adicionales como filtros y colores
  5. Haz clic en "Guardar" para crear el gráfico

Para personalizar tu dashboard:

  1. Ve a la sección "Dashboards" y selecciona uno existente o crea uno nuevo
  2. Haz clic en "Añadir Widget" para incluir gráficos a tu dashboard
  3. Arrastra y suelta los widgets para reorganizarlos
  4. Ajusta el tamaño de cada widget según tus preferencias
  5. Los cambios se guardan automáticamente

Para generar un informe:

  1. Ve a la sección "Informes" y selecciona "Nuevo Informe"
  2. Configura los modelos de datos que deseas incluir
  3. Personaliza los filtros y la configuración del informe
  4. Haz clic en "Generar" para crear el informe
  5. Una vez generado, puedes descargarlo en varios formatos
